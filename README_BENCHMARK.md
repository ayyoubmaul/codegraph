# Complete Codegraph Benchmark Package

## 🎯 Quick Summary
**Codegraph semantic code search is 41x more token-efficient than traditional grep-based approaches.**

- **Average tokens/query:** 92 (Codegraph) vs 3,891 (Traditional)
- **Cost per 100 queries:** $0.028 (Codegraph) vs $1.166 (Traditional)
- **Efficiency range:** 24x to 55x depending on query complexity
- **Monthly savings:** $113.84 (for 10,000 queries)

A: Tested on Rust, but principle applies to any language. The semantic search approach is language-agnostic.
